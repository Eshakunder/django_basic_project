# Django Basic Project

A foundational Django web application starter. This project is configured with a basic structure, including a custom app, SQLite database, and static file management.

##  Tech Stack
* **Framework:** [Django](https://www.djangoproject.com/)
* **Language:** Python 3.x
* **Database:** SQLite (Default)
* **Frontend:** Django Template Engine (HTML/CSS)

## 📁 Project Structure
```text
django_basic_project/
│── core/                  # Project configuration (settings.py, urls.py)
│── my_app/                # Main application folder (models, views, templates)
│── static/                # CSS, JavaScript, and Images
│── templates/             # Global HTML templates
│── manage.py              # Django command-line utility
└── requirements.txt       # Project dependencies
```
